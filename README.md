# Jenil Shingala — Portfolio

Personal portfolio website built with Next.js, Tailwind CSS, Framer Motion, and Spline 3D.

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Framer Motion** — animations
- **Spline** — interactive 3D scene
- **shadcn/ui** — component primitives
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
